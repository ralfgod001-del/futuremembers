# Quant Platform Phase 38

Phase 38 reorganizes the Web workspace for beginner-friendly use.

## New Capabilities

- Reworks the sidebar into a numbered workflow:
  - choose contract and historical data
  - manage watchlist
  - run the AkShare moving-average backtest
  - run advanced config files
  - adjust K-line indicators
  - inspect CTP monitor state
- Groups advanced data-source and fee settings behind collapsible panels.
- Adds task navigation buttons for `行情`, `回测`, `结果`, and `监控`.
- Reorders the main workspace so market chart, backtest metrics, equity curve,
  result tables, and CTP monitor appear in a clearer sequence.
- Updates page title state when a contract is selected.

## Notes

- No backend behavior changes in this phase.
- Existing element IDs are preserved so previous JavaScript behavior and tests
  continue to work.
